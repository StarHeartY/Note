# 五、技术实现建议

数据库标签字段：

```sql
category_tag
topic_tag
experience_tag
```

对应：

```text
category_tag
→ 客观基础标签

topic_tag
→ 内容主题标签

experience_tag
→ 主观体验标签
```

示例：

```yaml
category_tag:
- 广东
- 物化生
- 580-600分

topic_tag:
- 专业选择
- 计算机类

experience_tag:
- 后悔冲学校
- 浪费了分数
```
