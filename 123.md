## ✨ 它能做什么？

CalculatorX 不是普通计算器，而是一个**移动端的计算机代数系统（CAS）**。它直接处理符号表达式，输出精确的数学结果，而非近似小数。

### 🧠 符号运算 — 结果保留精确形式
不丢失数学结构，自动化简并保留 √2、π、e 等精确符号：

- 输入 $$\sqrt{8}$$ → 输出 $$2\sqrt{2}$$
- 输入 $$\sin(\pi/4)$$ → 输出 $$\frac{\sqrt{2}}{2}$$（弧度模式）
- 输入 $$e^{i\pi}$$ → 输出 $$-1$$

### ∫ 微积分 — 求导、积分、极限

- 求导：$$\frac{d}{dx}(x^3 + 2x) = 3x^2 + 2$$
- 定积分：$$\int_{0}^{1} x^2 \, dx = \frac{1}{3}$$
- 不定积分：$$\int \frac{1}{x} \, dx = \ln|x| + C$$
- 极限：$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

### 🔢 超大数计算 — 突破双精度极限
基于斯特林近似与对数变换，**O(1) 时间内**处理宇宙级数字：

- 支持最大值：$$10^{9 \times 10^{18}}$$（10 的 9×10¹⁸ 次方）
- 计算 $$1000000!$$（100 万阶乘）→ 瞬间输出 556 万位的完整结果
- 计算 $$\binom{10^9}{10^5}$$（十亿选十万）→ 秒出精确组合数

### 📐 三角与代数 — 完整函数覆盖
- 三角函数：$$\sin$$、$$\cos$$、$$\tan$$、$$\cot$$、$$\sec$$、$$\csc$$（支持角度/弧度切换）
- 反三角函数：$$\arcsin$$、$$\arccos$$、$$\arctan$$ 等
- 对数与指数：$$\ln$$、$$\log$$、$$\exp$$，支持任意底数
- 排列组合：$$P(n,k)$$、$$\binom{n}{k}$$，超大参数支持
- 多项式展开：$$(x+1)^3 = x^3 + 3x^2 + 3x + 1$$

### 📝 自然公式输入 — 像写在纸上一样
- 基于 MathLive 引擎，实时渲染 LaTeX 公式，所见即所得
- 键盘集成积分、求和、根号、分数等专用数学符号
- Shift 键切换副功能，单手操作流畅

### 🧩 历史记录 — 一键回填复用
- 自动保存每次计算的输入与输出（LaTeX 格式）
- 点击历史条目，瞬间回填到当前输入框，方便修改重算

## 🎯 为什么选择 CalculatorX？

- **双引擎驱动**：集成 Giac + SymEngine，兼顾符号精确度和数值性能
- **离线运行**：所有计算引擎本地化，无需联网，保护隐私
- **专业键盘**：专为触屏优化，Shift键切换副功能，输入流畅
- **开源免费**：GPL v3 协议，欢迎贡献
