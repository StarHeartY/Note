## ✨ 它能做什么？

| 功能           | 说明                           |
| ------------ | ---------------------------- |
| 🧮 **符号计算**  | 积分、求导、极限、化简，结果保留 √2、π、e 精确形式 |
| 🔢 **超大数计算** | 支持 10^10^19 级别的阶乘/组合数，瞬间出结果  |
| 📐 **角度/弧度** | 一键切换，三角函数自动适配                |
| 📜 **历史记录**  | 自动保存计算过程，点击即可回填复用            |
| 🌗 **深色模式**  | 跟随系统，护眼舒适                    |

## 🚀 快速开始

1. 从 [Releases](https://github.com/starhearty/calculatorX/releases) 下载 `.hap` 安装包
2. 使用 [DevEco Studio](https://developer.harmonyos.com/cn/develop/deveco-studio) 或鸿蒙手机安装
3. 打开即用，无需任何配置

> 如果你没有鸿蒙设备，可以查看下方的 [在线Demo](https://calcx.startyi.com/demo)（如有部署）

---

## 🎯 为什么选择 CalculatorX？

- **双引擎驱动**：集成 Giac + SymEngine，兼顾符号精确度和数值性能
- **离线运行**：所有计算引擎本地化，无需联网，保护隐私
- **专业键盘**：专为触屏优化，Shift键切换副功能，输入流畅
- **开源免费**：GPL v3 协议，欢迎贡献
