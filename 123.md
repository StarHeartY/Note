# CalculatorX 官方网站

[![Deploy to GitHub Pages](https://github.com/StarHeartY/CalcX-web/actions/workflows/deploy.yml/badge.svg)](https://github.com/StarHeartY/CalcX-web/actions/workflows/deploy.yml)

CalculatorX 官方网站源码。基于 React + TypeScript + Vite 构建，部署于 [calcx.startyi.com](https://calcx.startyi.com)。

## 技术栈

| 类别  | 技术                      |     |
| --- | ----------------------- | --- |
| 框架  | React 18 + TypeScript   |     |
| 构建  | Vite 6                  |     |
| 样式  | CSS Modules + CSS 自定义属性 |     |
| 部署  | GitHub Pages (Actions)  |     |

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build

# 预览构建产物
npm run preview
```

## 项目结构

```
CalcX-web/
├── index.html                    # Vite 入口
├── public/                       # 静态文件（原样复制到 dist/）
│   ├── CNAME                     # calcx.startyi.com
│   ├── help/index.html           # 使用帮助（App WebView 加载）
│   ├── privacy/index.html        # 隐私政策（App WebView 加载）
│   ├── agreement/index.html      # 用户协议（App WebView 加载）
│   ├── css/style.css             # 子页面共享样式
│   └── images/                   # 图片素材
├── src/
│   ├── styles/
│   │   ├── tokens.css            # 设计 Token（颜色 / 字体 / 间距）
│   │   ├── global.css            # 全局重置
│   │   └── animations.css        # 滚动动画
│   ├── hooks/
│   │   ├── useTheme.ts           # 深浅色切换
│   │   └── useScrollReveal.ts    # 滚动显示
│   ├── data/
│   │   └── content.ts            # 文案数据中心
│   ├── components/
│   │   ├── hero/                 # 首屏
│   │   ├── features/             # 功能特色
│   │   ├── gallery/              # 界面展示
│   │   ├── architecture/         # 技术架构
│   │   ├── download/             # 下载指引
│   │   ├── layout/               # Header / Footer
│   │   └── ui/                   # 通用组件
│   └── App.tsx                   # 根组件
├── .github/workflows/deploy.yml  # 自动部署
└── vite.config.ts
```

## 设计

Apple/Linear 风格，极简黑白配色 + 蓝色点缀，支持深浅色模式。

## 部署

推送到 `main` 分支后，GitHub Actions 自动构建并部署到 GitHub Pages。
