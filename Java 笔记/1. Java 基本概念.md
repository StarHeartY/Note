# Java 基本概念

## JDK, JRE, JVM

#### JDK:

Java标准开发包，它提供了编译、运行Java程序所需的各种工具和资源，包括Java编译器、Java运行时环境，以及常用的Java类库等。

#### JRE:

Java运行环境，用于解释执行Java的字节码文件。JVM
Java虚拟机，是JRE的一部分。负责解释执行字节码文件，是可运行java字节码文件的虚拟计算机

#### 区别联系：（问答题常考）

JDK包含JRE，JDK 和 JRE 中都包含 JVM。

**区别**：JDK除了包含jre还包含一些常用开发工具和基础类库

**联系**：JDK用于*开发，JRE用于运行java程序，JVM是java编程语言的核心并且具有平台独立性。
