### 🧮 计算过程

**第一步：排序服务时间**  
将输入的服务时间 `56 12 1 99 1000 234 33 55 99 812` 从小到大排序：  
`1, 12, 33, 55, 56, 99, 99, 234, 812, 1000`

**第二步：分配顾客到 2 个服务点 (s=2)**  
采用贪心策略，将排序后的顾客依次分配给当前累计服务时间最短的服务点。

- **服务点 1** 将依次服务：`1, 33, 56, 99, 812`
- **服务点 2** 将依次服务：`12, 55, 99, 234, 1000`

**第三步：计算总等待时间**

- **服务点 1 的等待时间总和**：
    
    - 第 1 位顾客等待：1
    - 第 2 位顾客等待：1 + 33 = 34
    - 第 3 位顾客等待：34 + 56 = 90
    - 第 4 位顾客等待：90 + 99 = 189
    - 第 5 位顾客等待：189 + 812 = 1001
    - 小计：1 + 34 + 90 + 189 + 1001 = **1315**
- **服务点 2 的等待时间总和**：
    
    - 第 1 位顾客等待：12
    - 第 2 位顾客等待：12 + 55 = 67
    - 第 3 位顾客等待：67 + 99 = 166
    - 第 4 位顾客等待：166 + 234 = 400
    - 第 5 位顾客等待：400 + 1000 = 1400
    - 小计：12 + 67 + 166 + 400 + 1400 = **2045**
- **总等待时间** = 1315 + 2045 = **3360**
    

**第四步：计算平均等待时间**  
平均等待时间 = 总等待时间 / n = 3360 / 10 = **336**
