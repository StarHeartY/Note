# 🌌 StartYi Personal Homepage (profile-page)

这是我的个人门户网站源代码，部署于 [startyi.com](https://startyi.com)。它作为一个极简且酷炫的导航中心，连接着我的博客、项目以及社交媒体。

## ✨ 视觉特性

- **故障风标题 (Glitch Effect):** 使用纯 CSS 实现的赛博朋克风格动态标题。
- **动态粒子背景:** 基于 `tsparticles` 实现的交互式星空连线效果，支持鼠标感知。
- **玻璃拟态卡片 (Glassmorphism):** 半透明磨砂玻璃质感的导航卡片。
- **3D 视差交互:** 引入 `vanilla-tilt.js`，使导航卡片随鼠标悬停产生物理倾斜感。

## 🛠️ 技术栈

- **前端:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
- **动画库:** [tsparticles](https://particles.js.org/), [vanilla-tilt.js](https://micku7zu.github.io/vanilla-tilt.js/)
- **字体:** [JetBrains Mono](https://www.jetbrains.com/lp/mono/) (通过 Google Fonts CDN 加载)
- **部署:** Cloudflare Pages (与 GitHub 自动同步)

## 📂 项目结构

```text
profile-page/
├── index.html          # 主页入口 (包含子域名导航逻辑)
├── about.html          # 个人详细简介

├── assets/             # 静态资源 (图片、视频)
├── css/
│   └── style.css       # 核心样式 (含故障风动画)
├── js/
│   └── main.js         # 交互逻辑 (粒子初始化、3D 效果)
└── README.md           # 本文件
```

## 🚀 快速开始

1. 克隆项目:

```Bash
git clone [https://github.com/你的用户名/profile-page.git]
```
(https://github.com/你的用户名/profile-page.git)
2. 本地运行:
直接在浏览器中打开 index.html，或使用 VS Code 的 Live Server 插件预览。

3. 自定义:

- 在 index.html 中修改 nav-card 的 href 链接到你的子域名。

- 在 style.css 中通过 :root 修改变量来更换主题色。
